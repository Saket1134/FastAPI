# 🏥 FastAPI Patient Management API

A fully functional **FastAPI-based REST API** to manage patient records, including BMI calculation and health verdict.

---

## Features

- Create a patient  
- View all patients  
- View a specific patient by ID  
- Update patient details  
- Delete patient  
- Automatic **BMI calculation**  
- Health verdict (Underweight, Normal, Obese)  
- Sort patients by height, weight, or BMI  

---

## Tech Stack

- **FastAPI**
- **Python**
- **Pydantic**
- **Uvicorn**
- **JSON (file-based storage)**

---

## Perform CRUD Operation

-**C=Create  
Method=POST**  
-**R=Read  
Method=GET**  
-**U=Update  
Method=PUT**  
-**D=Delete  
Method=DELETE**
